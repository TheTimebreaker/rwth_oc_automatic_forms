This software is intended for members of the organic chemistry department of the RWTH Aachen University.

It can be used to more easily fill in forms for different analysis methods (e.g. for the NMR department, for the Mass department, etc. ).

Find the homepage [here](https://thetimebreaker.github.io/rwth_oc_automatic_forms)
